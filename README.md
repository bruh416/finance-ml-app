# Finance ML App

## Overview
This app allows users to upload financial datasets (e.g., Kragle), fetch real-time stock data from Yahoo Finance, and run machine learning models to analyze financial trends.

## Requirements
- streamlit
- pandas
- numpy
- scikit-learn
- matplotlib
- plotly
- yfinance
